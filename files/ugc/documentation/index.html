A) HTML Structure - top

Magazine is XHTML Strict template. It's made with Twitter Bootstrap CSS framework. Almost every layout styling is covered by Bootstrap unless some layout that generally Bootstrap can't do. This theme is a fluid layout with two columns. The general template structure is the same throughout the template. Here is the general structure.

Header

<header id="header"> contains the site logo and navigation bar. Editing #logo you can easily change the home link and the logo image.

Navigation

Site navigation is found in <nav class="navbar navbar-inverse clearfix nobot">. To create main menu items, add list elements with class nav. To add a submenu item create child list item with class "dropdown-menu" and don't forget to add "dropdown" class to parent list item.

B) CSS Structure - top

The following css files are used in every template html and php files. They can be found in the HEAD section of the page. Style.css is the template css file, the others are twitter bootstrap and the styles require for each template feature. 

<link rel='stylesheet' id='magz-style-css'  href='style.css' type='text/css' media='all' />
<link rel='stylesheet' id='swipemenu-css'  href='css/swipemenu.css' type='text/css' media='all' />
<link rel='stylesheet' id='bootstrap-css'  href='css/bootstrap.css' type='text/css' media='all' />
<link rel='stylesheet' id='bootstrap-responsive-css'  href='css/bootstrap-responsive.css' type='text/css' media='all' />
<link rel='stylesheet' id='simplyscroll-css'  href='css/jquery.simplyscroll.css' type='text/css' media='all' />
<link rel='stylesheet' id='jPages-css'  href='css/jPages.css' type='text/css' media='all' />
<link rel='stylesheet' id='rating-css'  href='css/jquery.rating.css' type='text/css' media='all' />
<link rel='stylesheet' id='ie-styles-css'  href='css/ie.css' type='text/css' media='all' />
<link rel='stylesheet' id='Roboto-css'  href='http://fonts.googleapis.com/css?family=Roboto' type='text/css' media='all' />

  • style.css - Main CSS for template, all main styles are here.
  • css/
    • swipemenu.css - Default styling swipe mobile menu
    • bootstrap.css - Default styling for all basic HTML elements by Twitter Bootstrap
    • bootstrap.reponsive.css - Default styling to control HTML elements in various screen width by Twitter Bootstrap
    • jquery.simplyscroll.css - Default styling of scrolling news

Home page (index.html or magazine2.html) requires 1 more css file to control the slide featured on the page.

<link rel='stylesheet' id='flexslider-css'  href='css/flexslider.css' type='text/css' media='all' />

Gallery page requires 1 more css file to control the photo lightbox.

<link rel='stylesheet' id='pretty-photo-css'  href='css/prettyPhoto.css' type='text/css' media='all' />

Home page (index.html or magazine2.html) requires 4 more js files to control the slide and carousel featured on the page.

Gallery page requires more js file to control the photo lightbox.

Contact page requires more js file to set a Google map.

F) Uploading Template - top

Best way to upload template is if you use FTP program. I strongly recomment you to use Filezilla

Filezilla is available for PC and Mac and it's easy to use. We assume that you already have hosting, so open filezilla, login and find "public_html" folder. On some servers it is called "www" folder...that's folder where you need to upload all files
